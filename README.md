## ✨ 特性

- **🎯 零依赖** - 所有项目均为单文件HTML，无需npm/webpack/任何构建工具
- **🎨 纯前端** - 100%客户端实现，无需后端服务器
- **📱 响应式** - 支持桌面端和移动端浏览
- **🎮 即开即用** - 双击HTML文件即可在浏览器中运行
- **🔥 高质量** - 包含WebGL、物理模拟、算法可视化等高难度项目

## 🚀 使用方法

### 方式1: 直接打开

1. 下载或克隆本仓库
2. 用浏览器打开 `index.html` 浏览所有项目
3. 点击任意项目卡片即可在新标签页中打开

### 方式2: 启动本地服务器

```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 📖 技术栈

所有项目均使用以下纯前端技术实现：

- **HTML5** - 页面结构
- **CSS3** - 样式与动画
- **Vanilla JavaScript** - 核心逻辑
- **Canvas 2D API** - 2D图形渲染
- **WebGL/WebGL2** - 3D图形与GPU计算
- **Web Audio API** - 音频合成与分析
- **MediaDevices API** - 摄像头访问

## 🎯 项目亮点

### 🔥 高难度实现

- **Navier-Stokes流体模拟** - GPU加速的实时流体动力学
- **Ray Marching渲染器** - SDF距离场实时3D渲染
- **神经网络可视化** - 前向/反向传播完整实现
- **Boids群体行为** - 复杂的群体智能模拟
- **反应扩散系统** - Gray-Scott模型生成图案
- **量子电路模拟器** - 量子门与复数振幅计算
- **Lisp解释器** - Tokenizer → Parser → Evaluator完整流程

### 🎨 视觉效果

- **Aurora极光背景** - 三层径向渐变动画
- **毛玻璃效果** - `backdrop-filter: blur(10px)`
- **卡片悬浮动画** - `transform: translateY(-6px) scale(1.02)`
- **光泽扫过效果** - 伪元素渐变动画
- **渐变文字** - `background-clip: text`

### ⚡ 性能优化

- **GPU加速** - WebGL计算密集型任务
- **RequestAnimationFrame** - 流畅60fps动画
- **离屏Canvas** - 双缓冲减少闪烁
- **Web Workers** - 多线程计算（部分项目）
- **空间分区** - 四叉树/八叉树优化碰撞检测
